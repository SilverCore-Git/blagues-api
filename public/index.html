Comment utiliser Blagues l'API ?

Voici quelques exemples d'utilisation de l'API de Blagues pour récupérer des blagues :

1. Obtenir une blague par son identifiant

Pour récupérer une blague avec un identifiant spécifique, vous pouvez utiliser l'URL suivante :

https://blagues.api.silverdium.fr/api?id=joke_id

Remplacez joke_id par l'ID de la blague que vous souhaitez.

2. Obtenir une blague par type

Vous pouvez aussi obtenir une blague selon un type spécifique avec cette URL :

https://blagues.api.silverdium.fr/api?type=nom_du_type

Remplacez nom_du_type par un type valide de blague.
(types : dark, limit, global, dev, beauf et blondes. )

3. Obtenir une blague aléatoire

Pour obtenir une blague au hasard, utilisez l'URL suivante :

https://blagues.api.silverdium.fr/api?random=true

4. Obtenir une blague aléatoire d'un type spécifique

Pour obtenir une blague aléatoire d'un type précis, utilisez cette URL :

https://blagues.api.silverdium.fr/api?type=nom_du_type&random=true

Encore une fois, remplacez nom_du_type par un type de blague spécifique.
